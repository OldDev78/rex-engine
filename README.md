<h1 align="center">ReX Engine</h1>

**[ReX Engine](https://www.redotengine.org) is a cross-platform game engine for creating 2D and 3D games.** It offers a set of tools to simplify game development. Games can be exported to desktop (Linux, macOS, Windows), mobile (Android, iOS), web, and consoles.

</div>

## Project Status

ReX is **actively under development**.

While this public repository may appear quiet at times, development has **not stopped**. A significant portion of ongoing work is currently being done in a **private development repository**. This allows the team to iterate freely on core systems, architectural changes, and experimental features before they are ready for public release.

### Why keep development private?
Because ReX is developed by a small team, we need the space to build, test, and refine core systems before they can be used or adopted by much larger organizations with significantly more resources. This approach protects the project’s long-term direction and sustainability.

We appreciate your patience and interest in ReX, and we’ll continue to share updates, releases, and milestones here as development progresses.


## Redot Engine

Looking for a stable engine you can bring your current Godot projects over to? Checkout the [Redot Engine Repo](https://github.com/Redot-Engine/redot-engine) where we strive to maintain compatibility with current Godot versions while fixing bugs and making improvements that won't break your current projects.
